J-query No-conflict: when we work on web , it may use many diff-2 frameworks where it may possible another framework use $ sign to start jquery code. so it may conflict jquery code with others.
to handle this kinds of problem we can create jquery no-conflict method and use by specific keyword.


Jquery method chaining: jquery provide facility to call list of methods in a chain uaing chaining.

exa: 
    $.noConflict(true);
	jQuery(document).ready(function(){
	jQuery("#btn").click(function(){
	
		jQuery("#dd").css("background-color","yellow").fadeOut(3000).
		fadeIn(3000).slideUp(2000).slideDown(3000)
		
	
	})
})


Jquery on method: this method can be use bind event with element, it is similar to eventlistener which capable to bind multiple events to element.

<script type="text/javascript">
  
    $.noConflict(true);
	jQuery(document).ready(function(){
	jQuery("#txt").on({"keypress":function(){
		$(this).css("background-color","yellow");
	},
	"keyup":function(){
		$(this).css("background-color","gray");
	},
	"blur":function(){
		$(this).css("background-color","green");
	}
	
	})	
	})



	</script>
